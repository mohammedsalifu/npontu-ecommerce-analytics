# Project Title
Analyzing Customer Behavior for E-commerce Insights

# Project Overview
Brief description of the business problem.

# Objectives
Analyze customer purchasing behavior
Identify churn patterns
Engineer meaningful customer features
Build a predictive Random Forest model
Generate actionable business recommendations

# Dataset
Include:
Source: Kaggle
Number of records: 350
Number of features: 11

# Methodology
Data Cleaning
Exploratory Data Analysis
Feature Engineering
Data Preprocessing
Random Forest Modeling
Evaluation and Insights

# Results
Model Accuracy : 98.57%

# Key Findings
Insight 1: Customer Inactivity Drives Churn

Days Since Last Purchase emerged as the strongest predictor of churn.

Recommendation:
Implement automated retention campaigns for customers who have not made purchases within 20–30 days.

Insight 2: Customer Satisfaction Matters

Satisfaction Level was the second most important factor.

Recommendation:
Monitor customer feedback continuously and address dissatisfaction before customers disengage.

Insight 3: High-Value Customers Need Protection

High-value customers and customers with high spending contribute significantly to business revenue.

Recommendation:
Create loyalty and VIP retention programs targeted at top spenders.

Insight 4: Discounts Influence Retention

Discount Applied was an important feature.

Recommendation:
Use personalized discounts to re-engage customers who show signs of inactivity.
